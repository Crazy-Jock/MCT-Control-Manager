# MCT Control Manager
Данный проект предназначен для составления прогноза, автоматического расчёта и контроля за временем наземного обслуживания ВС по критерию MCT (Minimum Connection Time — минимальное время обслуживания).
## Features
- Создание рейсов: выбор класса ВС, тип ВС, пассажировместимость, номер рейса. Возможность работать сразу с несколькими рейсами одновременно.
- Ввод планового и ожидаемого времени прибытия ВС и автоматические рассчеты времени выполнения на каждую операцию.
- Ввод фактического времени прибытия ВС и ввод фактического времени на каждую операцию.
- Автоматическое сравнение ожидаемого и фактического MCT.
- Вывод текстового отчета о выполнении обслуживания и отклонениях от плана.
- Вывод временной сети Петри для просмотра полного цикла обслуживания с цветовыми индикациями.
- Сохранение рейсов в отдельные файлы.
- Запись и просмотр статистики по MCT по всем или каждому классу за определенный период.
- Внесение новых типов ВС для автозаполнения MCT и пассажировместимости.
